# 冰滴咖啡機控制器 v3.3 旗艦靜音版 — 專案重梳理

> **狀態**: 已全面移除 CYD / S3 舊架構。
> **核心**: ESP32-WROOM-32D + 一體化 OLED 面板 + LR7843 MOSFET + 2S 鋰電系統。

## 🎨 顏色分碼配線指南 (Color-Coded Wiring)

為了方便組裝，我們將接線分為五大顏色類別。請在組裝時儘量選用對應顏色的杜邦線。

| 起點 | 終點 | 顏色建議 | 說明 |
| :--- | :--- | :--- | :--- |
| 電池組 (2S) | 2S BMS (B+/B-/BM) | 🔴/⚫ | 電池核心保護連接 |
| BMS (P+) | 升壓充電板 (Out+) | 🔴 | Type-C 充電入口 |
| BMS (P+) | XL4015 (IN+) | 🔴 | **總電源輸入** (8.4V) |
| XL4015 (OUT+) | 麵包板 5V 軌 | 🔴 | **系統主電力** (5.0V) |
| XL4015 (OUT-) | 麵包板 GND 軌 | ⚫ | **系統共地** |

## ⚠️ 組裝與調試注意事項 (必讀)

1.  **電壓先行**: 在連接任何零件到 ESP32 之前，**請務必先將電池接上 XL4015 並用萬用表調整螺絲，確保輸出端為準確的 5.0V**。
2.  **共地 (GND)**: 所有的黑色地線（電池負極、ESP32 GND、MOSFET GND、降壓模組 OUT-）都必須連通，否則訊號會失效。
3.  **MOSFET 接法**: 
    *   `VIN+ / VIN-`: 接 8.4V 電池。
    *   `OUT+ / OUT-`: 接馬達。
    *   `PWM / GND`: 接 ESP32 訊號。
    *   **優點**: 馬達運轉時，電流不經過 ESP32，非常安全。
4.  **電阻分辨**: 
    *   10K 色環: 棕黑黑紅棕。
    *   30K 色環: 橘黑黑紅棕。
