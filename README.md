# ☕ 冰滴咖啡機控制器 (Cold Drip Coffee Controller) v3.3

> **硬體版本**: v3.3 旗艦靜音版
> **核心架構**: ESP32-WROOM-32D + 一體化 OLED 面板 + LR7843 MOSFET + 2S 鋰電 Type-C 充電

本專案旨在打造一個具備專業 UI、極致靜音、且可攜帶（Type-C 充電）的冰滴咖啡自動化控制系統。

---

## 📋 系統規格

| 類別 | 規格 |
| :--- | :--- |
| **目標產量** | 600ml / 6-12小時 |
| **主控核心** | **ESP32 DevKit V1 (WROOM-32D)** |
| **操作/顯示** | **1.3" OLED + 編碼器一體化面板 (128x64)** |
| **靜音驅動** | **LR7843 MOSFET 模組 × 2** (高電平觸發) |
| **電力儲存** | **18650 鋰電池 × 2** (串聯 2S / 7.4V) |
| **電力管理** | **BMS 保護板** + **Type-C 升壓充電** + **XL4015 穩壓 (5V)** |

### 核心腳位定義 (GPIO Mapping)
| ESP32 引腳 | 類型 | 連接零件 | 功能描述 |
| :--- | :--- | :--- | :--- |
| **GPIO 21** | SDA | OLED 數據線 | I2C Data |
| **GPIO 22** | SCL | OLED 時鐘線 | I2C Clock |
| **GPIO 32** | Input | 編碼器 CLK | 旋轉訊號 A |
| **GPIO 33** | Input | 編碼器 DT | 旋轉訊號 B |
| **GPIO 35** | Input | 編碼器 SW | 旋鈕按下 (按鈕) |
| **GPIO 34** | ADC | 30K/10K 分壓點 | **電池電壓監控** |
| **GPIO 25** | Output | MOSFET 1 | 水泵啟動 (HIGH=ON) |
| **GPIO 26** | Output | MOSFET 2 | 氣泵啟動 (HIGH=ON) |

## 🛠️ 組裝說明

1.  **電源安全**: 首次接線前，務必先將 XL4015 降壓模組調整至 **5.0V**，再連接 ESP32。
2.  **共地要求**: 所有模組的 GND 必須連通。
3.  **電量校準**: 系統使用 30K (R1) 與 10K (R2) 分壓電阻。若電壓顯示不準，請微調 `Config.h` 中的阻值參數。
4.  **低溫環境**: 本系統支援 4°C 冰箱運作，但請務必將電池與電路板放置於密封盒內並加入乾燥劑。
