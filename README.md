# ☕ 冰滴咖啡機控制器 (Cold Drip Coffee Controller)

> ESP32-CYD 驅動的 600ml / 6小時自動化冰滴咖啡系統
> **全 5V 統一架構** — 無需升壓模組，電路精簡可靠

## 📋 專案概述

| 項目 | 規格 |
|------|------|
| **目標產量** | 600ml / 6小時 |
| **配置** | 左右並列雙槽結構（右側入水、左側萃取） |
| **控制器** | ESP32 CYD (ESP32-2432S028) |
| **電源架構** | 全 5V 統一供電（無升壓模組） |
| **運作模式** | 完全離線，無需 WiFi |
| **核心邏輯** | 間歇滴灌（預設每 60 秒啟動 2.5 秒） |
| **完成後處理** | 自動 Air Purge 吹氣清掃管路 (30 秒) |

## 🔧 BOM 表 (物料清單)

| 類別 | 項目規格 | 用途 |
|------|----------|------|
| 控制器 | ESP32 CYD (ESP32-2432S028) | 系統大腦與資訊顯示 (5V) |
| 水泵 | Kamoer NKP 蠕動幫浦 (**6V** / BPT管) | 精準抽水，5V 供電可運作 |
| 氣泵 | **5V** 小型 Air Pump | 萃取後 Air Purge 吹氣清掃 |
| 電源 | Eneloop Pro AA 電池 x 4 (串聯) | 穩定、耐低溫電力來源 (4.8~5.8V) |
| 降壓件 | DC-DC 降壓模組 (降至 **5V**) | 統一供應所有元件 |
| 驅動件 | D4184 MOSFET 模組 **x 2** | 分別控制水泵 (GPIO27) 與氣泵 (GPIO22) |
| 監測件 | 分壓電阻 (100kΩ / 10kΩ) | 電池電壓百分比監測 |
| 佈水器 | ⌀60 × H10mm 多孔圓盤 | 確保水均勻進入粉層 |
| 管件 | 三通接頭 (3mm) × 1 | 水泵/氣泵管路匯流 |
| 管件 | 單向閥 (3mm) × 3 | 防止逆流，確保流向正確 |
| 耗材 | 3mm 食品級矽膠管 (FDA/NSF) | 水路與氣路管線 |

> ⚡ **相較 12V 方案省掉了 MT3608 升壓模組**，降低成本、減少 EMI 干擾、提升 10-15% 電池續航。

## ⚡ 接線圖

![5V 統一架構配線圖](docs/wiring_diagram.png)

### GPIO 分配

| GPIO | 功能 | 方向 | 接往 |
|------|------|------|------|
| **GPIO27** | 水泵控制 | OUTPUT | D4184 MOSFET #1 SIG |
| **GPIO22** | 氣泵控制 (Air Pump) | OUTPUT | D4184 MOSFET #2 SIG |
| **GPIO35** | 電壓監測 (ADC1_CH7) | INPUT | 分壓電阻中間點 |
| 5V | 電源輸入 | - | DC-DC 降壓 VOUT+ |
| GND | 共地 | - | 所有模組共地匯流排 |

### 接線重點 — 5V 統一架構

```
電池 (+) ──┬── DC-DC 降壓 → 5V ─┬─ ESP32 CYD (5V 供電)
           │                     ├─ D4184 #1 VIN → Kamoer NKP 6V 水泵
           │                     └─ D4184 #2 VIN → 5V Air Pump 氣泵
           │
           └── 分壓電阻 (100kΩ + 10kΩ) → GPIO35 (電池電壓 ADC)

ESP32 GPIO27 → D4184 #1 SIG (水泵開關)
ESP32 GPIO22 → D4184 #2 SIG (氣泵開關)
所有 GND 統一接至共地匯流排
```

> 💡 **不需要 MT3608 升壓模組！** 所有動力元件統一由 5V 降壓輸出供電。

## 📐 機構設計

三層垂直結構：控制模組（頂部）→ 佈水器+粉槽（中層左側）→ 容器區（底部）

![正面機構圖](docs/mechanical_front_view.png)

### 總體尺寸

| 維度 | 尺寸 | 說明 |
|------|------|------|
| **總高度** | ~345mm | 控制模組 55 + 佈水器 10 + 粉槽 60 + 容器 220 |
| **總寬度** | 180mm | 左容器 80 + 間距 20 + 右容器 80 |
| **總深度** | 80mm | 容器深度 |
| **粉槽** | ⌀60 × H60mm | 圓柱體，~170ml，位於左側萃取區正上方 |
| **佈水器** | ⌀60 × H10mm | 多孔圓盤，確保水均勻滴入粉層 |
| **容器** | 80×80×220mm × 2 | 左=萃取接液，右=進水 |

> 控制模組置於頂部，方便操作螢幕、更換電池與維護線路。

### 管路系統

![管路系統圖](docs/plumbing_schematic.png)

```
萃取模式: 進水區 → 水泵 → 單向閥A → 三通 → 單向閥B → 佈水器 → 粉槽 → 萃取區
清掃模式: 氣泵 → 單向閥C → 三通 → 單向閥B → 佈水器 → 粉槽 (吹出殘留水)
```

| 單向閥 | 位置 | 用途 |
|--------|------|------|
| **A** | 水泵出口 → 三通 | 防止 Air Purge 時空氣逆流回水泵 |
| **B** | 三通 → 佈水器 | 確保流向一律往佈水器方向 |
| **C** | 氣泵出口 → 三通 | 防止正常抽水時水灌入氣泵 |

## 🖥️ 操作介面

### 主控頁面
- **狀態指示**: STANDBY / BREWING / PAUSED / PURGING / COMPLETE / LOW BATT!
- **即時數據**: 已萃取量、已耗時、剩餘量、剩餘時間
- **進度條**: 萃取/Air Purge 完成百分比
- **電池監測**: 電壓與百分比顯示
- **控制按鈕**: START / PAUSE / RESUME / SKIP / STOP

### 設定頁面
- **滴灌間隔**: 10~300 秒，每次調整 ±10 秒
- **幫浦持續時間**: 500~5000 毫秒，每次調整 ±100 毫秒
- **流量預估**: 即時計算 ml/min 與總預估量

## 🛡️ 安全機制

| 保護類型 | 觸發條件 | 動作 |
|----------|----------|------|
| 低電量停機 | 電壓 < 4.0V | 自動停止水泵/氣泵，切換至 LOW BATT 狀態 |
| 低電量警告 | 電壓 < 4.4V | 電池圖示轉為琥珀色 |
| 電壓平滑 | - | 8 點滑動平均，避免瞬間干擾 |
| 啟動保護 | 啟動時 V < 4.0V | 拒絕啟動萃取 |

## 📝 開發筆記

- **5V 統一架構**: 省掉升壓模組，所有元件共用 5V 供電，降低 EMI 與功耗
- **離線設計**: 完全不需要 WiFi，適合冰箱冷藏環境
- **防水考量**: 電子艙建議密封並內置乾燥劑包，防止冷藏環境結露
- **局部刷新**: UI 採用局部更新策略，降低閃爍並節省電力
- **滑動平均**: 電池電壓使用 8 點滑動平均，過濾 ADC 雜訊
- **流量補償**: 6V 泵在 5V 下流量降低，透過延長 ON 時間補償

## ⚠️ 注意事項

- 首次使用前，建議先校準幫浦流量（透過設定頁面調整 ON 時間）
- 確保分壓電阻焊接穩固，ADC 讀數偏差會影響電量計算
- PETG 基座需確保密封性，防止冷凝水滲入電子艙
- Eneloop Pro 在低溫環境下容量會略減，實際續航時間可能縮短
- 幫浦管材務必使用 **FDA/NSF 認證食品級矽膠管**
